# Быстрый запуск федеративного обучения

## Самый простой способ

Запустите файл `start_training.bat` - он откроет 4 окна (1 сервер + 3 клиента):

```cmd
start_training.bat
```

## Что произойдет:

1. Откроется окно с сервером - подождите пока увидите "Flower ECE: gRPC server running"
2. Откроются 3 окна с клиентами - каждый подключится к серверу
3. Начнется обучение - вы увидите прогресс в окнах
4. После завершения модели сохранятся в `flower_server\models\`

## Проверка результатов:

После завершения обучения (обычно 2-5 минут):

```powershell
Get-ChildItem flower_server\models\*.pt
```

Должны быть файлы:
- `active_model.pt` - финальная обученная модель
- `model_round_N.pt` - модели после каждого раунда

## Альтернативный способ (если bat файл не работает):

Откройте 4 отдельных окна PowerShell и выполните команды из `MANUAL_START_INSTRUCTIONS.md`

## Конфигурация:

Текущие настройки (можно изменить в `start_training.bat`):
- Раундов: 5
- Эпох на клиенте: 3
- Batch size: 32
- Learning rate: 0.001

## Устранение проблем:

### "Port already in use"
- Закройте все предыдущие процессы Python
- Или измените порт в `start_training.bat`: `set SERVER_PORT=8081`

### Клиенты не подключаются
- Убедитесь, что сервер запущен первым
- Подождите 5-10 секунд перед запуском клиентов
- Проверьте, что все используют `localhost:8080`

### Нет моделей после завершения
- Проверьте логи в окнах сервера и клиентов на ошибки
- Убедитесь, что данные в папке `output\` существуют
- Попробуйте увеличить `NUM_ROUNDS` до 10
